Support for all modern web browsers

There are many different ways to structure an HTML page, but there are only a few essential tags that every page should have from the start. Keeping these crucial tags while excluding the non-essential ones will help save time when creating a new HTML document. It is even easier when you can copy and paste the code instead of having to write it out. The default template featured has everything you need, including:

  • <!DOCTYPE>
  • <html>
  • <head>
  • <body>

<!DOCTYPE> - this indicates what version of HTML to use (HTML 5 is the most recent iteration). Keep in mind that this is not an HTML tag but rather a piece of information for a browser—a must when creating a new HTML page.

<html> - root of the HTML document, excluding the DOCTYPE tag. The HTML tag will contain all the information for your page.

<head> - contains all the metadata for the web page, which essentially helps tell online bots and browsers what type of information is on the page. Having no head tag will make it difficult for search engines to index your page and for people to find a site.

<body> - has all the page's text, images, pictures, and visual properties. Think of it as having everything you see on the page, while the head tag is everything about it.

Apart from all the essential tags needed as described above, there are many other tags to consider for your next project. Some of these include:

  • <title>
  • <meta>
  • <h1>-<h6>
  • <link>

<title> - how the link for your site will show up on search engine results and tabs on browsers. The title is an essential tag as it will be the first thing a user is shown about your site and will be why someone clicks on it rather than another site. It should also explain the webpage's main topic, which will help with SEO and getting the website in front of the right eyes.

<meta> - apart from the title tag (also considered a meta tag), the meta tags will be the most important for SEO and getting indexed high in the search results for search engines such as Google, Bing, DuckDuckGo, etc. Multiple meta tags should be incorporated on every site, including description, robots, canonical, viewport, and social media tags, to name a few. You can find a complete list of all the essential meta tags here.

<h> - define headings throughout a webpage, often signaling essential issues and the corresponding information with each topic. There are 6 HTML headings, the first being the most important and the sixth being the least important. Use these tags to highlight and structure your page to show users and web crawlers the most critical information. Think of the h tag as the name of each chapter in a book. It will summarize what you will be reading before you start.

<link> - the link tag will fetch an external source and is most commonly used to add style sheets and favicons. These tags do not have an element inside. Instead, it is empty and contains attributes only.

Of course, there are many other tags to use for a modern webpage, but not all of them are required. Make sure to include the essentials and slowly add new tags as your site gets more extensive and has more content.

SEO, otherwise known as search engine optimization, is necessary for getting a webpage discovered on the internet with trillions of other pages. When people search for stuff online, the first page is usually where all the eyeballs will be. Anything beyond the first page of results will be invisable, as not many people will browse past the first page. Good SEO will push your site to the top of search results on search engines. Good SEO can be obtained by:

  1. Good title tag
  2. Informative description tag
  3. Unique/Niche content
  4. Have good keywords throughout your page
  5. Add images and videos related to the content of the page
  6. Make sure loading times are fast
  7. Add links to other high-quality pages related to yours
  8. Optimize images
  9. Make site responsive to all device screen sizes
  10. Pick a short domain name
  11. Use HTTPS over HTTP
  12. Include a sitemap if your site has many pages
  13. Try and avoid duplicate content
